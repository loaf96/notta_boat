These documents are used to scrape getmyboat.com to collect data on their listings. 'imonna_boat' is a python script that will collect all of the data that is shown on their boat-rentals page and puts it into a csv file for easier viewing, and future use. 

This task was performed for a client who wished to be able to easily display and identify the information contained within the csv file.

Once ran, 'imonna_boat' will create a .csv file titled: 'scrprman.csv' located in the same folder as the location of the python script.

Modules that were used/will be needed to execute the python code were:
	- splinter
	- BeautifulSoup
	- pandas
